# Sanskriti.Traditional

**Sanskriti.Traditional** is a culture-first e-commerce platform designed to connect Indian artisans with modern, conscious buyers.

Unlike generic marketplaces, this platform focuses on:
- Authentic handcrafted products
- Artisan visibility and storytelling
- Trust-first user experience
- Frictionless checkout and post-purchase flow

## Problem Statement
Handcrafted products often suffer from lack of authenticity, poor trust signals, and weak digital experiences. Buyers hesitate, and artisans remain invisible.

## Solution
Sanskriti.Traditional bridges this gap by combining cultural storytelling with modern e-commerce UX.

## Key Features
- Culture-first landing experience
- Product browsing without forced login
- Secure checkout with delayed authentication
- Artisan, Heritage, and Journal sections
- Wishlist, Order Tracking, and Account Dashboard
- Mobile-first responsive design

## Tech Stack
- HTML, CSS, JavaScript
- Responsive UI design
- Netlify deployment
